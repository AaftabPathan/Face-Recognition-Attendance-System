2. STEP-BY-STEP DEPLOYMENT INSTRUCTIONS (Setup kaise karein)
--------------------------------------------------------------------------------

STEP 1: Database (MySQL) Setup Karna
--------------------------------------
1. Apne computer par **XAMPP Control Panel** open kijiye.
2. Apache aur MySQL dono ke samne waale **`Start`** button par click kijiye. (Dono green ho jayenge).
3. Apne Chrome browser mein ye URL open kijiye: http://localhost/phpmyadmin/
4. phpMyAdmin mein left side panel mein **`New`** par click kijiye:
   * Database Name: **`face_attendance_db`** daaliye.
   * **`Create`** button par click kar dijiye.
   * (Aapka blank database ban gaya hai).
5. Ab aapko table aur sample data import karna hai:
   * `face_attendance_db` database par click kijiye, phir top menu se **`Import`** tab par click kijiye.
   * **Choose File** click karke is file ko select kijiye aur page ke bilkul niche jaakar **`Import/Go`** click kijiye:
     📂 `backend/database/schema.sql` (Isse saare 10 tables ban jayenge).
   * Import complete hone ke baad, dubara **`Import`** click kijiye aur is dusri file ko select karke import kijiye:
     📂 `backend/database/seed.sql` (Isse demo students, teachers aur reports load ho jayenge).

⭐ AUTOMATIC ALTERNATIVE (Chutkiyon mein database setup karne ke liye):
Agar aap manual phpMyAdmin import nahi karna chahte, toh:
* Apne VS Code terminal mein `backend` directory ke andar bas ye command run kar dijiye:
  `node database/init.js`
  (Ye automatic script khud-b-khud database banayegi, tables banayegi aur saara data 2 seconds mein load kar degi!).


STEP 2: Connection Credentials Configure Karna
------------------------------------------------
1. Apne project ke **`backend/`** folder ke andar **`.env`** file ko open kijiye.
2. Wahan line no. 3 par **`DB_PASSWORD=`** hai:
   * Agar aap XAMPP default use kar rahe hain, toh password khali (blank) hi chodiye: `DB_PASSWORD=`
   * Agar aapne local MySQL par koi password set kiya hai (jaise `root` ya kuch aur), toh use wahan enter kijiye: `DB_PASSWORD=aapka_password`
3. Save kar dijiye.


STEP 3: Backend Server Start Karna
------------------------------------
1. Apne VS Code mein ek Terminal open kijiye aur `backend` directory ke andar jaiye:
   ```bash
   cd backend
   ```
2. dependencies install karne ke liye (agar pehli baar run kar rahe hain):
   ```bash
   npm install
   ```
   *(Note: Maine aapke system par ye pehle se hi install kar diya hai!).*
3. Backend dev server ko start kijiye:
   ```bash
   npm run dev
   ```
   *Terminal mein message aayega:*
   `✅ MySQL Database connected successfully!` aur `🚀 Express Application running on port 3000`.
   *(Is terminal window ko chalta rehne dein, isko close mat karna).*


STEP 4: Frontend Angular Server Start Karna
---------------------------------------------
1. VS Code ke Terminal panel ke right side par **`+` (Plus)** button par click kijiye ek **Naya Terminal tab** open karne ke liye.
2. Naye terminal mein `frontend` directory ke andar jaiye:
   ```bash
   cd frontend
   ```
3. dependencies install karne ke liye (agar pehli baar run kar rahe hain):
   ```bash
   npm install
   ```
   *(Note: Maine aapke system par ye pehle se hi install kar diya hai!).*
4. Angular frontend application ko chalu kijiye:
   ```bash
   npm start
   ```
5. Compilation complete hone ke baad (pehli baar compile hone mein 1-2 minutes lag sakte hain), terminal par ye message aayega:
   `✔ Compiled successfully.` aur niche **`Local: http://localhost:4200/`** ka link dikhega.

🎉 BINGO! Ab apne Chrome browser mein **`http://localhost:4200`** open kijiye!

5. HOW IT WORKS UNDER THE HOOD (AI aur Computer Vision Kaise kaam karta hai)
--------------------------------------------------------------------------------
Aapke college project external examiner ya presentation ke liye kuch important technical points:

1. **Client-Side Biometric Scanning:** Hamara Angular app **`face-api.js`** (built on top of TensorFlow.js) library ka use karta hai. SSD MobileNet V1 neural model ke zariye ye webcam frames ko 30+ FPS par browser ke andar process karta hai.
2. **Biometric Face Embeddings:** Video frame se face detect karke ye face landmark coordinates (68 points shape) nikalta hai, aur use **128-dimensional array of floats** (biometric embeddings vector) mein convert karta hai.
3. **Euclidean Distance Formula:** Face matching ke liye ye current webcam face coordinate array ($A$) aur database mein save coordinate array ($B$) ke bich Euclidean Distance mathematical algorithm lagata hai:
   $$Distance = \sqrt{\sum_{i=1}^{128} (A_i - B_i)^2}$$
   * **Threshold (0.60):** Agar distance **0.60 ya usse kam** hai, toh use match (Present) mana jata hai. Distance jitna kam hoga, biometric match confidence utna hi 100% accurate hoga!
4. **Dual-Verification Model:** Frontend browser 30 FPS par matching karke smooth boxes draw karta hai aur backend Express API secure dual-verification mathematical calculations check karke database mein attendance entry log karta hai. Isase project highly secure aur robust banta hai.
